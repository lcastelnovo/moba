# Changelog

## [0.1.0] - 2026-03-07

### Added
- Initial release
- Rails Engine with isolated namespace
- React 19 + TypeScript frontend via Superglue
- Resource DSL for defining admin resources
- Dynamic CRUD (index, show, new, edit) based on resource definitions
- Field types: text, email, select
- DataTable with TanStack Table
- Redux Toolkit store with Superglue reducers
- Sidebar navigation with registered resources
